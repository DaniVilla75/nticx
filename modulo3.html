🌐 La Web y el Hipertexto

¿Alguna vez te preguntaste cómo funciona internet? Todo empieza con conceptos fundamentales que hoy son parte de nuestra vida cotidiana.

📡 Digitalización de la Información

La digitalización es el proceso de convertir información analógica (fotos en papel, cassettes, libros) a formato digital (bits: 0 y 1). En Argentina 2026, esto es clave:

📄

Documentos Digitales

El DNI ahora es digital (Mi Argentina), las facturas son electrónicas (AFIP), y las escuelas usan SIPE.

Ej: El SIPE (Sistema Integral de Permanencia Escolar) de la PBA digitaliza asistencias y calificaciones.
🎵

Medios Digitales

Spotify, YouTube Music y Flow reemplazaron los CDs. El streaming es digitalización en tiempo real.

Ej: Flow (Telecom Argentina) permite ver TV en cualquier dispositivo.
🏛️

Gobierno Digital

El Estado argentino digitaliza trámites: turnos online, pagos digitales, expedientes electrónicos.

Ej: TAD (Trámites a Distancia) de la PBA para gestiones escolares.

🔗 Conceptos Clave de la Web

Tocá cada tarjeta para descubrir la definición:

🌐

World Wide Web (WWW)

Tocá para ver la definición →

¿Qué es?

Sistema de información global que usa internet para acceder a documentos interconectados mediante hipervínculos. Inventada por Tim Berners-Lee en 1989.

💡 Dato: ¡No es lo mismo que Internet! La WWW es un servicio que usa Internet.

📜

HTML

Tocá para ver la definición →

HyperText Markup Language

Lenguaje de marcado que define la estructura de las páginas web. Usa "etiquetas" para indicar títulos, párrafos, imágenes, etc.

<h1>Hola Mundo</h1>
<p>Esto es un párrafo</p>
📡

HTTP/HTTPS

Tocá para ver la definición →

Protocolo de Transferencia

Conjunto de reglas que permite la comunicación entre navegador y servidor. HTTPS es la versión segura (con candadito 🔒).

💡 Si no tiene HTTPS, ¡no ingreses datos personales!

🔗

Hipervínculo

Tocá para ver la definición →

Link o Enlace

Elemento clickeable que te lleva a otro documento, página o recurso. Es la base de la navegación web.

💡 El color azul y el subrayado son convenciones, no reglas estrictas.

🎬

Hipermedia

Tocá para ver la definición →

Texto + Multimedia

Extensión del hipertexto que incluye imágenes, audio, video y animaciones interconectados.

💡 TikTok, Instagram y YouTube son hipermedia pura.

📧

Correo Electrónico

Tocá para ver la definición →

e-mail

Sistema de comunicación digital que permite enviar mensajes, archivos y multimedia a través de internet. Usa protocolos SMTP, POP3 e IMAP.

💡 En Argentina, muchos usan @gmail, @hotmail o correos institucionales @bue.edu.ar

💬 Otros Servicios de la Web

💬

Chat

Comunicación en tiempo real por texto. Evolucionó desde el IRC de los 90 hasta WhatsApp, Discord y Telegram.

Ej: Los grupos de WhatsApp de padres de la escuela son "chats".
🎙️

Podcast

Programas de audio digitales que podés escuchar on-demand. Crecieron explosivamente en Argentina.

Ej: "El Destape", "Cienfuegos", "La Cruda" - podcasts argentinos populares en 2026.
📡

RSS

Really Simple Syndication. Sistema para suscribirse a actualizaciones de sitios web sin visitarlos manualmente.

Ej: Los lectores de noticias como Feedly agregan RSS de Clarín, La Nación, etc.

🕳️ Deep Web y Dark Web

Internet es mucho más grande de lo que vemos. Cuando hacés una búsqueda en Google, solo accedés a una pequeña porción de toda la información existente. El resto está en la Deep Web.

🧊

Surface Web (Web Superficial)

Todo lo que podés encontrar con un buscador normal (Google, Bing). Está indexado públicamente y cualquiera puede acceder.

Ejemplos:
• Sitios de noticias (Clarín, La Nación)
• Redes sociales (Instagram, X/Twitter)
• Wikipedia, YouTube, blogs
• Representa ~10% de toda la web
🕳️

Deep Web (Web Profunda)

Contenido que NO está indexado por los buscadores convencionales. No es ilegal: simplemente requiere acceso directo, login o permisos especiales.

Ejemplos:
• Bases de datos académicas (JSTOR, SciELO)
• Contenido de redes sociales privado
• Expedientes judiciales digitales
• Repositorios universitarios (Repositorio CONICET)
• Representa ~90% de toda la web
⚫

Dark Web (Web Oscura)

Una pequeña parte de la Deep Web que requiere software especial (como Tor) para acceder. Anonimiza la identidad del usuario.

Usos legítimos:
• Periodismo de investigación (filtraciones)
• Comunicación en países con censura
• Whistleblowers (denunciantes)

⚠️ También se usa para actividades ilegales:
• Venta de drogas, armas, datos robados

⚠️ Importante: Deep Web ≠ Dark Web

Mucha gente confunde estos términos. La Deep Web es simplemente contenido no indexado (como tu correo de Gmail o tu historial de búsqueda). La Dark Web es una subcapa que requiere software especial y se asocia con anonimato total.

💡 Dato: Cada vez que entrás a tu cuenta de banca online (Home Banking de tu banco), estás en la Deep Web. ¡No es peligroso, es privado!

🧊🕳️⚫ Analogía: El Iceberg de Internet

🧊
Surface Web
~10% — Todo lo indexable por Google
🕳️
Deep Web
~90% — Contenido no indexado (bases de datos, intranets, contenido privado)
⚫
Dark Web
Una fracción de la Deep Web — Requiere Tor, anonimato total

📂 Tipos de Buscadores

🏛️

Buscadores Jerárquicos

Los más comunes. Usan arañas para indexar todo internet y algoritmos para ordenar resultados.

Ejemplos:
• Google (líder mundial)
• Bing (Microsoft)
• Yahoo! Search
• DuckDuckGo (privacidad)
📁

Directorios Web

Organizan sitios en categorías manuales. Hoy casi en desuso, pero fueron fundamentales en los 90.

Ejemplos:
• DMOZ (Open Directory Project) - ya cerró
• Yahoo! Directory - descontinuado
• Directorios temáticos educativos
🎯

Buscadores Especializados

Se enfocan en un área específica: académica, médica, legal, de compras, etc.

Ejemplos:
• Google Scholar (académico)
• PubMed (medicina)
• Wolfram Alpha (computacional)
• MercadoLibre (compras - 🇦🇷)
🔀

Metabuscadores

No tienen base de datos propia. Consultan varios buscadores a la vez y combinan resultados.

Ejemplos:
• Dogpile (combina Google, Yahoo, Yandex)
• MetaCrawler
• Skyscanner (vuelos - metabuscador turístico)
• Indeed (empleo - metabuscador laboral)
• Trivago (hoteles)

✅ Cómo Evaluar si una Fuente es Confiable

Encontrar información es fácil. Evaluarla es la habilidad realmente importante. Antes de usar una fuente para una tarea, un trabajo o para formar tu opinión, hacete estas 5 preguntas:

👤

1. Autoría

¿Quién escribió esto? ¿Es una persona, institución u organismo identificable? ¿Tiene credenciales o experiencia en el tema?

✅ Bien: un artículo firmado por un investigador del CONICET.
⚠️ Dudoso: una publicación anónima en un grupo de Facebook.
📅

2. Actualidad

¿Cuándo se publicó o actualizó por última vez? En temas de tecnología, salud o ciencia, la información vieja puede estar obsoleta.

✅ Bien: una nota de 2025 sobre vacunas, con fecha visible.
⚠️ Dudoso: una "noticia" sin fecha que circula hace años por WhatsApp.
🎯

3. Objetivo

¿Para qué se escribió? ¿Busca informar, vender, opinar o convencer? Un sitio que vende algo puede exagerar beneficios.

✅ Bien: un artículo informativo de un medio reconocido.
⚠️ Dudoso: una web que vende "productos milagrosos" y también "informa" sobre sus beneficios.
🔗

4. Respaldo / Fuentes

¿Cita de dónde sacó los datos? ¿Otros medios o fuentes serias dicen lo mismo? La triangulación (comparar varias fuentes) es clave.

✅ Bien: un artículo que enlaza al estudio científico original.
⚠️ Dudoso: afirmaciones grandes sin ningún link ni referencia.
🌐

5. Dominio y diseño

El tipo de dominio da pistas: .gob.ar (gobierno), .edu.ar (educación), .org (organizaciones). Un diseño descuidado, con muchos errores de ortografía o publicidad invasiva, es señal de alerta.

✅ Bien: argentina.gob.ar, conicet.gov.ar, una universidad.
⚠️ Dudoso: sitios con URLs raras, llenas de pop-ups y banners parpadeantes.
🕵️

Herramientas de verificación

Ante una noticia dudosa, usá verificadores de hechos (fact-checking) antes de compartirla.

En Argentina:
• Chequeado.com
• Reverso (Agencia AFP)
Internacional:
• Google Fact Check Explorer
• Búsqueda inversa de imágenes (Google Lens, TinEye)

🤖 Evolución de la Web: 1.0 a 4.0

La web no siempre fue como la conocemos hoy. Pasó por etapas que transformaron cómo interactuamos con la información.

1991 - 2004

🌐 Web 1.0 - La Red Pasiva

La web "de solo lectura". Los usuarios consumían contenido estático creado por webmasters. Páginas HTML fijas, sin interactividad.

Características: Sitios estáticos, guestbooks, contadores de visitas, geocities.
Ejemplo argentino: Los primeros sitios de diarios como Clarín.com en los 90, portales como ElOtroLado.net.
2004 - 2010

👥 Web 2.0 - La Red Participativa

La web "social". Los usuarios pasan de consumidores a creadores de contenido. Surgen las redes sociales, blogs, wikis.

Características: Blogs, wikis, redes sociales, tagging, AJAX, APIs.
Ejemplos argentinos: Taringa! (foro/comunidad argentina), Facebook, Twitter/X, YouTube, Wikipedia.
2010 - 2020

🧠 Web 3.0 - La Web Semántica

La web "inteligente". Las máquinas entienden el significado de los datos. Uso de metadatos, ontologías, datos vinculados (Linked Data).

Características: RDF, ontologías, motores de recomendación, asistentes de voz.
Ejemplos: Siri, Google Assistant, recomendaciones de Netflix/Spotify, motores de búsqueda semántica.
2020 - 2026

🤖 Web 4.0 - La Web de la IA

La web "activa y predictiva". La inteligencia artificial personaliza la experiencia, predice necesidades y actúa proactivamente.

Características: IA generativa, personalización extrema, web ubícua (IoT), metaverso, asistentes conversacionales.
Ejemplos argentinos 2026: ChatGPT/Claude para tareas escolares, asistentes de banca digital (Mercado Pago, Ualá), recomendaciones de IA en PedidosYa, diagnóstico médico con IA en hospitales públicos de PBA.

🌎 Cibercultura y Sociedad Red

Internet no es solo tecnología: es cultura, economía, política y sociedad. Entender estos conceptos nos hace ciudadanos digitales más conscientes.

🧠 Conceptos Fundamentales

🌐

Cibercultura

Conjunto de prácticas, valores, símbolos y normas que surgen del uso de internet y las tecnologías digitales. Es la "cultura de internet".

Ejemplos argentinos 2026:
• Memes políticos (mileistas, opositores)
• Lenguaje digital: "crack", "fachero", "re contra"
• Streamers argentinos: Coscu, Ibai, Spreen
• Comunidades de Discord de escuelas secundarias
🕸️

Sociedad Red

Concepto del sociólogo Manuel Castells. Describe una sociedad donde la estructura organizativa es una red de redes, no piramidal.

En Argentina:
• Organizaciones sociales que usan WhatsApp para convocatorias
• Redes de trueque digital (Trueque+)
• Colectivos de arte digital (NFTs argentinos)
📊

Sociedad de la Información

Sociedad donde la información es el recurso económico principal. La producción, distribución y consumo de información generan valor.

En Argentina:
• Economía del conocimiento (software, biotech)
• MercadoLibre como plataforma de información comercial
• Sistema de información educativa (SIPE, SINIDE)
🎓

Sociedad del Conocimiento

Va más allá de la información: se valora el conocimiento aplicado, la creatividad, la innovación y el aprendizaje permanente.

En Argentina:
• Programas como Argentina Programa (educación tech)
• Universidades con MOOCs (Coursera, edX)
• Comunidades de aprendizaje en GitHub/StackOverflow

⚠️ Alfabetización Mediática: ¡Cuidado!

No todo lo que leemos en internet es verdad. En 2026, la desinformación es un problema real:

  • 🚨 Fake News: Noticias falsas que circulan por WhatsApp y redes. Verificá en Chequeado.com o Reverso.
  • 🚨 Deepfakes: Videos manipulados con IA. En Argentina ya hubo casos de políticos.
  • 🚨 Burbuja de filtros: El algoritmo te muestra solo lo que coincide con tus opiniones. ¡Salí de tu zona de confort!
  • 🚨 Phishing: Mensajes falsos que roban datos. Nunca des tu contraseña por mail.

📖 Wikipedia: ¿confiable o no?

Wikipedia es la enciclopedia colaborativa más grande del mundo. Cualquiera puede editarla, lo cual genera una pregunta clásica en clase: ¿se puede usar como fuente?

✅

Para qué SÍ sirve

Excelente punto de partida: te da una visión general del tema, fechas, datos básicos y, lo más importante, enlaces a las fuentes originales al final del artículo.

Usala para entender de qué se trata un tema y para encontrar links a estudios, libros o sitios oficiales citados en la sección "Referencias".
⚠️

Para qué NO conviene citarla directamente

En trabajos académicos formales, no se cita "Wikipedia dice..." porque el contenido puede cambiar en cualquier momento y cualquiera puede editarlo (aunque hay revisión comunitaria).

Mejor práctica: leé el artículo de Wikipedia, identificá la fuente original en las referencias, y citá esa fuente original.
🔍

Revisá la "Discusión" e "Historial"

Cada artículo de Wikipedia tiene pestañas de "Historial" (todos los cambios hechos) y "Discusión" (debates entre editores). Mirar esto te muestra si un tema es controvertido o si hay ediciones recientes sospechosas.

Si un artículo tiene un cartel de "Este artículo necesita referencias adicionales", tomalo con cautela extra.

©️ Derechos de Autor y Licencias Creative Commons

Cuando usás una imagen, texto, música o video de internet para un trabajo escolar (o para subir a redes), hay reglas sobre qué podés usar y cómo. Esto se llama propiedad intelectual.

🔒

Todos los derechos reservados

Es lo que se asume por defecto si no se indica nada. El autor es el único que decide cómo se usa su obra. Usarla sin permiso (por ejemplo, una foto de un fotógrafo profesional) puede traer problemas legales.

🌍

Creative Commons (CC)

Sistema de licencias que permite a los autores compartir su trabajo bajo ciertas condiciones, sin perder todos los derechos. Muy usado en fotos, música y artículos educativos.

Las licencias más comunes:
• CC BY — Podés usar y modificar, citando al autor.
• CC BY-SA — Igual, pero si modificás debés compartir bajo la misma licencia.
• CC BY-NC — Uso permitido solo si NO es comercial.
• CC0 / Dominio Público — Sin restricciones, uso totalmente libre.
🖼️

Bancos de imágenes libres

Para trabajos escolares, usá bancos de imágenes con licencias abiertas en vez de "robar" la primera foto de Google.

Recomendados:
• Wikimedia Commons
• Unsplash / Pexels
• Pixabay
Tip: en Google Imágenes → Herramientas → Derechos de uso → "Creative Commons"

📝 Cómo citar una fuente correctamente

Citar es dar crédito a quien creó la información que usaste. Además de ser una cuestión de honestidad, evita el plagio (presentar el trabajo de otro como propio).

✍️ Formato básico de cita (estilo simplificado para la escuela)

Apellido, Nombre del autor (o nombre del sitio/institución).
"Título del artículo o página".
Nombre del sitio web.
Fecha de publicación o de consulta.
URL completa.

Ejemplo real:

Ministerio de Educación de la Provincia de Buenos Aires.
"Plan Provincial de Conectividad Educativa".
abc.gob.ar.
Consultado el 15 de junio de 2026.
https://www.abc.gob.ar/...

💡 Para trabajos más formales, herramientas como Google Docs o Word tienen generadores de citas automáticos (formato APA, MLA, etc.), y sitios como Cite This For Me ayudan a armarlas a partir de la URL.
